# PVF Delivery Map — current state

Updated: 2026-07-06

## Status: BUILT + LOCALLY VERIFIED, not yet published

Inspired by the Digitl Pastures cold-email drip (their "customers should
see where you deliver" pitch) — built in-house instead.

## What exists

- Full app in `docs/`: Leaflet map with both zones shaded (Rochester slate
  blue, Buffalo orange), farm pin (Leicester), Brighton Farmers Market pin
  (Sundays, Brighton High School lot), ZIP checker with per-route next
  delivery date + Thursday 11:59 PM deadline, market cards, "not on the
  map" email-list CTA, shop/planner footer buttons.
- Zone data: 85 ZCTA polygons (43 Rochester + 42 Buffalo), simplified to
  ~136 KB, generated by `builder/build_zones.py`. All 85 ZIPs matched a
  polygon.
- Embed loader `docs/embed.js` (same pattern as the order planner).
- Verified locally 2026-07-06: no console errors; date logic correct
  (Rochester Jul 11 / deadline Jul 9 — July 2nd-Saturday exception works;
  Buffalo Jul 18 / deadline Jul 16); ZIP checker verified for Rochester
  hit (14618), Buffalo hit (14052), and miss (14411); mobile layout stacks.

## Known limits

- ZIP polygons are 2010-vintage ZCTAs (OpenDataDE); fine at this zoom.
- The map publishes the full zone ZIP lists by nature — intended (that is
  the point of the page), but the Buffalo demographic rationale stays in
  the shared brain, never in this repo.
- Initial map bounds include a lot of Lake Ontario (wide Rochester–Buffalo
  –Leicester triangle); acceptable, revisit if Brian wants tighter framing.
